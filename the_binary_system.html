The binary system

in hacking movies or documentarys you sometimes see people with 10011 1001110 10110 10101110 on their screens

That is the language of the binary system.

The binary system is a numerical system that uses only two digits, 0 and 1, and is fundamental in computer science.

It represents numbers using powers of 2, with each digit's place value indicating a power of 2.

For example, the decimal number 13 is represented as "1101" in binary, where each digit's place value corresponds to a power of 2.
